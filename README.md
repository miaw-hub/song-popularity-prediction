 # **Song Popularity Prediction**
 A Machine Learning regression model that predicts a song's popularity score (0-100) based on Spotify audio features like energy, danceability, and acousticness."
 
 **Key FeaturesFeature Analysis:** 
 Evaluates which audio characteristics (e.g., loudness, tempo) most influence a song's success.
 **Regression Pipeline:** 
 Uses scaled numerical data to predict popularity with high precision.
 **Interactive Prediction:**
 Users can input specific song metrics to see a predicted popularity score.
